Learn by building a user creation form with dynamic dropdowns and validation.

## What You'll Build

A form that collects user information and executes a workflow to create the user.

## Prerequisites

-   [Installation complete](/guides/installation)
-   At least one workflow created (try [First Workflow](/tutorials/first-workflow) first)

## Create the Form

<Steps>

1. Navigate to **Forms** → Click **+** button

2. Fill in form details:

    - **Name**: "Create User Account"
    - **Description**: "Provision a new user"
    - **Linked Workflow**: Select your workflow (e.g., `create_user`)
    - **Scope**: Organization

3. Click **Create**

</Steps>

## Add Fields

<Steps>

1. **From Workflow Parameters** (recommended):

    - Drag workflow parameters from left panel
    - Fields auto-configure with correct types

2. **Or use Field Types**:

    - Drag field type (Text, Email, Select, etc.)
    - Configure manually

3. Arrange fields by dragging

</Steps>

## Configure Field Settings

Click any field to edit:

### Basic Settings

-   **Label**: Display text
-   **Placeholder**: Hint text in empty field
-   **Help Text**: Additional guidance
-   **Required**: Enforce field completion
-   **Default Value**: Pre-populated value

### Validation

For text fields:

```javascript
// Email validation (auto-applied for email type)
// Custom: check length
context.value.length >= 3;
```

For dropdowns, use data providers:

-   Select **Data Provider** instead of static options
-   Choose provider (e.g., `get_departments`)

## Add Visibility Rules

Show/hide fields conditionally:

```javascript
// Show "Manager Email" only if "Is Manager" is checked
context.field.is_manager === true;

// Show "Department" only if "Employee Type" is "full-time"
context.field.employee_type === "full-time";
```

## Test the Form

<Steps>

1. Click **Save** in form builder

2. Click **Preview** or navigate to **Forms** list

3. Click your form → **Execute**

4. Fill out fields and click **Submit**

5. View result or execution logs

</Steps>

## Use Data Providers

Create dynamic dropdowns with data providers:

1. Create `workspace/data_providers/departments.py`:

```python
from bifrost import data_provider, ExecutionContext

@data_provider(
    name="get_departments",
    description="List of departments",
    cache_ttl_seconds=600
)
async def get_departments(context: ExecutionContext):
    return [
        {"label": "Engineering", "value": "eng"},
        {"label": "Sales", "value": "sales"},
        {"label": "Support", "value": "support"}
    ]
```

2. In form builder, select field → **Data Provider** → `get_departments`

3. Save and test - dropdown auto-populates
