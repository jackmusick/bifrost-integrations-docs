Bifrost is an open-source workflow automation platform for MSPs and integration services. It bridges complex enterprise systems with user-friendly automation.

## What Bifrost Solves

**Problem**: Organizations struggle with manual processes, disconnected systems, complex integrations, and multi-tenant requirements.

**Solution**: Bifrost provides:
- **Democratized Automation**: Non-developers use workflows via forms
- **System Integration**: Connect Microsoft Graph, HaloPSA, custom APIs
- **Auto-Scaling**: Azure serverless (Functions, Storage, Key Vault)
- **Data Isolation**: Multi-tenant with organization scoping
- **Enterprise Security**: Auth, encryption, audit logging

## Core Concepts

### Workflows
Python async functions decorated with `@workflow`:

```python
@workflow(name="create_user", description="Create new user")
@param("email", "email", required=True)
async def create_user(context, email: str):
    return {"user_id": "123"}
```

### Forms
UI for workflows with:
- Multiple field types
- Data provider integration
- Visibility rules
- Real-time validation

### Data Providers
Dynamic options for dropdowns:

```python
@data_provider(name="get_departments")
async def get_departments(context):
    return [
        {"label": "Engineering", "value": "eng"},
        {"label": "Sales", "value": "sales"}
    ]
```

### Discovery System
Workflows auto-register on startup:
1. Scans `/workspace/workflows/` and `/workspace/data_providers/`
2. Imports Python files
3. Executes decorators
4. Registers metadata
5. Exposes via REST API

## Multi-Tenancy

- **Organization Isolation**: Strict data separation
- **Org-Scoped Secrets**: Stored as `{org_id}--secret_name`
- **Context Scoping**: Workflows execute with org context
- **Per-Org Configuration**: Independent settings

## Security

- **Authentication**: Azure AD, function keys
- **Authorization**: Role-based access control (RBAC)
- **Data Isolation**: Organization-level separation
- **Secret Management**: Azure Key Vault with encryption
- **Audit Logging**: All actions logged with user context
- **OAuth Flow**: Secure OAuth 2.0 token management
