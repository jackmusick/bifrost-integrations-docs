Agents are AI-powered surfaces in Bifrost that plan multi-step work, call tools you expose as workflows, and produce auditable runs. Each agent has a system prompt, a tool list, and a budget that bounds how much work it can do per run.

Agents can be invoked interactively from Chat, triggered autonomously on a schedule or event, or delegated to from another agent. Their runs are stored alongside workflow executions for review, tuning, and ROI analysis.
